```ts
export interface EmbeddedChunk {

  embedding: number[];  // 1536-dim vector from ada-002

  speaker?: string;     // e.g. "Agent" or "User" — present on transcript chunks
}
```

**Pure utility functions:**

```ts
export function cosineSimilarity(a: number[], b: number[]): number
```
- Compute dot product of a and b divided by (magnitude(a) * magnitude(b))
- Handle zero-vector case: return 0 if either magnitude is 0

```ts
export function topK(query: number[], chunks: EmbeddedChunk[], k: number): EmbeddedChunk[]
```
- Score each chunk with cosineSimilarity(query, chunk.embedding)
- Sort descending by score
- Return first k chunks (no similarity threshold — always return best matches per user decision)

**Static KB store with lazy init:**

```ts
// Module-level state
let kbChunks: EmbeddedChunk[] | null = null;
let kbInitPromise: Promise<void> | null = null;

export async function initKB(rawTexts: string[]): Promise<void>
```
- Called once at server startup (or lazily on first request)
- Imports embedBatch from ./embeddings
- Sets kbChunks to embedded chunks (no meetingId, no speaker)
- Uses kbInitPromise to prevent double-init: if kbInitPromise is set, await it and return

```ts
export async function getKBChunks(): Promise<EmbeddedChunk[]>
```
- Returns kbChunks if already initialized
- If not, throws an Error("KB not initialized — call initKB() first")
- Do NOT auto-initialize here; initialization is the caller's responsibility
