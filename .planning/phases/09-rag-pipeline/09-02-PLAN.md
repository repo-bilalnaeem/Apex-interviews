**retrieve.ts — Full replacement:**

```ts
import 'server-only';
import { embed } from './embeddings';
import { initKB, getKBChunks, getTranscriptChunks, topK, EmbeddedChunk } from './vectorStore';
import { knowledgeBase } from './knowledgeBase';

// Lazy-init sentinel — runs once per server process
let kbReady = false;

async function ensureKB(): Promise<void> {
  if (kbReady) return;
  await initKB(knowledgeBase);
  kbReady = true;
}

export async function getRelevantDocs(query: string, meetingId?: string): Promise<string> {
  await ensureKB();

  const queryVector = await embed(query);
  const TOP_K = 5;

  const kbChunks = await getKBChunks();
  const kbResults = topK(queryVector, kbChunks, TOP_K);

  let transcriptResults: EmbeddedChunk[] = [];
  if (meetingId) {
    const txChunks = getTranscriptChunks(meetingId);
    if (txChunks && txChunks.length > 0) {
      transcriptResults = topK(queryVector, txChunks, TOP_K);
    }
  }

  const parts: string[] = [];

  if (kbResults.length > 0) {
    parts.push('Interview coaching knowledge:\n' + kbResults.map(c => c.text).join('\n'));
  }

  if (transcriptResults.length > 0) {
    parts.push(
      'Relevant transcript excerpts from this interview:\n' +
      transcriptResults.map(c => c.speaker ? `${c.speaker}: ${c.text}` : c.text).join('\n')
    );
  }

  return parts.join('\n\n');
}
```

IMPORTANT: The function signature changes from `getRelevantDocs(query: string): string` (sync) to `getRelevantDocs(query: string, meetingId?: string): Promise<string>` (async). The route.ts caller in Plan 03 will await this.

Add a new step at the END of the `meetingsProcessing` inngest function, AFTER the "save-summary" step.

The new step name: `"embed-transcript"`

What it does:
1. Takes `transcriptWithSpeakers` (already computed earlier in the function — array of `{ text, start_time, end_time, type, user: { name } }`)
2. Chunks by speaker turn: each item in transcriptWithSpeakers becomes one chunk. If a turn's text exceeds ~200 tokens (roughly 800 chars), split into fixed 800-char windows. For most turns this won't trigger.
3. For each chunk, prefix the text with the speaker name from `item.user.name`:
   - Determine if speaker is agent or user. The `agents` table was already queried — use `agentSpeakers` to check if the speaker is an agent. If the speaker's name matches an agent, label as "Agent". Otherwise label as "User".
   - Format: `"Agent: {text}"` or `"User: {text}"`
4. Collect all formatted strings into `chunkTexts: string[]`
5. Import `embedBatch` from `@/lib/chatbot/embeddings` and `storeTranscriptChunks`, `EmbeddedChunk` from `@/lib/chatbot/vectorStore`
6. Call `const vectors = await embedBatch(chunkTexts)`
7. Build `EmbeddedChunk[]` from chunkTexts + vectors + meetingId + speaker label
8. Call `storeTranscriptChunks(event.data.meetingId, embeddedChunks)`

The step returns void (no return value needed).
